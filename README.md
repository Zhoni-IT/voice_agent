# Voice-to-Voice AI Agent for Smart Gas Station 🚗⛽

Прототип интеллектуального голосового ассистента для цифровой заправки. Система способна вести многошаговый диалог с водителем, распознавать голосовые команды (STT), автоматически вызывать бизнес-инструменты (Tool Calling) на основе намерений пользователя и озвучивать финальный ответ реалистичным нейросетевым голосом (TTS).

## 🚀 Технологический стек
* **Backend Framework:** FastAPI (Python)
* **LLM Orchestration:** LangChain / LangCore
* **Inference API:** Groq Cloud (Llama 3.3 70B Versatile)
* **Speech-to-Text (STT):** Groq Whisper-large-v3 (обработка аудио за доли секунды)
* **Text-to-Speech (TTS):** Edge-TTS (Neural voice generation)

## 🛠️ Функционал агента (Tool Use)
Агент в режиме реального времени анализирует контекст и самостоятельно активирует нужную Python-функцию:
1. `check_balance` — проверка остатка литров топлива и денежных средств.
2. `create_order` — автоматическое формирование заказа на заправку с извлечением параметров (тип топлива, количество литров).
3. `get_history` — просмотр истории последних транзакций.
4. `change_limits` — изменение дневных или разовых лимитов на заправку.

## 📦 Архитектура проекта
Проект реализует полный цикл **Voice-in -> Voice-out**:
1. Клиент отправляет аудиофайл (.mp3/.wav) на эндпоинт `/voice-input`.
2. Модель **Whisper-large-v3** транскрибирует аудио в текст.
3. **LangChain-агент** анализирует текст, обращается к памяти диалога, при необходимости вызывает соответствующий инструмент (Tool) и формирует лаконичный текстовый ответ.
4. **Edge-TTS** превращает текст в потоковое аудио и возвращает готовый `.mp3` файл обратно пользователю.

## 🔧 Инструкция по запуску

1. Клонируйте репозиторий:
```bash
git clone [https://github.com/your_username/voice_agent.git](https://github.com/your_username/voice_agent.git)
cd voice_agent
```

2. Создайте и активируйте виртуальное окружение:
```bash
python -m venv venv
source venv/Scripts/activate  # Для Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Создайте файл .env в корневой директории и добавьте ваш ключ:
```bash
GROQ_API_KEY=your_groq_api_key_here
```

5. Запустите сервер:
```bash
uvicorn main:app --reload
```

После запуска интерактивная документация API (Swagger UI) доступна по адресу: http://127.0.0.1:8000/docs.
